💰 Payment Planner - Personal Finance Management App

📱 Overview

Payment Planner is a comprehensive mobile application designed to help users take control of their personal finances. The app enables users to track expenses, manage budgets, set savings goals, and gain insights into their spending habits.

🎯 Features

✅ Authentication
- User registration with email and password
- Secure login system
- Session management

📊 Dashboard
- Financial overview at a glance
- Monthly salary vs expenses tracking
- Real-time savings calculation
- Budget breakdown visualization
- Savings progress with percentage tracking
- Months-to-goal estimator

💸 Expense Management
- Add, edit, and delete expenses
- Categorize expenses (Food, Transport, Entertainment, etc.)
- Date and time tracking for each expense
- Date range filtering for expense history
- Total expense calculation per period
- Optional photo attachment for expenses

🏷️ Category Management
- Create custom expense categories
- Edit and delete categories
- Color-coded categories for easy identification
- Pre-loaded default categories

🎯 Savings Goals
- Set monthly savings targets
- Track progress towards goals
- Visual progress indicators
- Remaining amount calculation

📸 Photo Capture
- Take photos directly from the app
- Attach photos to expense entries
- View photos from expense details
